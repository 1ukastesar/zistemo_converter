1. Stáhnout a nainstalovat python (https://www.python.org/downloads/)
    - pro kontrolu napsat do cmd python
2. Stáhnout a nainstalovat pip (https://bootstrap.pypa.io/get-pip.py)
    - Stáhnout get-pip.py
    - Přes cmd se dostat do složky, kde se nachází get-pip.py a spustít instalaci pomocí python get-pip.py

3. Cmd navigation se do složky s projektem ZistemoConverter
4. Napsat pip install -r requirements.txt (instalace potřebných knihoven)
5. Napsat python -m venv venv (instalace virtualního prostředí pythonu pro daný projekt)
    - pro změnu adresy a portu v app.py řádek 38 (.py soubor se dá editovat i v poznámkovým bloku)
6. Napsat .\venv\Scripts\activate (aktivace venv)
7. python app.py (spuštění)
